
# PDF viewer

upload pdfs to mongoDB and can read them after upload.


## Run Locally

Clone the project

```bash
  https://github.com/Nikeshchaudhary52494/pdfViwer.git
```

Go to the project directory

```bash
  cd pdfViwer
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
